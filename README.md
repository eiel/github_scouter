# GithubScouter

あなたのGitHub戦闘力を計測します。

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'github_scouter'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install github_scouter

## Usage

    $ github_scouter [github ID]
